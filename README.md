# KK_SensibleH
This is a BepInEx plugin for Koikatsu, with substantial amount of features specifically for VR mode of [MainGameVR](https://github.com/mosirnik/KK_MainGameVR/).
Can be used outside of VR.

The aim is to:
* Give the majority of H actions a semi-automatic mode
* Improve lackluster aspects of H
* Enhance features of [MainGameVR](https://github.com/mosirnik/KK_MainGameVR/) plugin.

**In development**, due to lack of documentation and hard dependencies **isn't recommended for users**.

For those that feel confident, grab whole package with dependencies/modified versions of plugins and preferably your VR headset.
